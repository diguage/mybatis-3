= 前言

https://mybatis.org/mybatis-3/[MyBatis^] 是一款优秀的持久层框架，它支持自定义 SQL、存储过程以及高级映射。MyBatis 可以通过简单的 XML 或注解来配置和映射原始类型、接口和 Java POJO（Plain Old Java Objects，普通老式 Java 对象）为数据库中的记录。

学习和理解优秀开源框架的设计思想，是提升自我设计能力的捷径。源码分析是学习开源框架设计设想的不二法门。本文档着重学习和记录对 MyBatis 源码分析的点滴知识。希望点滴知识，汇聚成汪洋大海。
